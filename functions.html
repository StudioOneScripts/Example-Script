Functions Description Examples
.closeHeaderTracks() Closes all open headers, Tempo, Markers, etc
.getInstrumentRack() Returns the native synth rack object
Tracks
.getTracks() Returns an array of all tracks
const Tracks = S1.getTracks();
for (const track of Tracks){
  print(track.name)
}
.getSelectedTracks() Returns an array of all selected tracks
.getTrackList() Returns the tracklist object
.normalizeTrackNames() Performs case normalizaton on all track names
.normalizeTrackNames() Performs case normalizaton on all track names
.dbToFloat() Comnverts a decibel value to floating point
.floatToDb() Converts a floating point value to decibel
Channels
.getMixer() Returns the mixer object
.getMasterBus() Returns the master bus object
.getChannels() Returns an array of all mixer channels const Channels = S1.getChannels();
for (const channel of Channels){
   print(channel.label)
}
.getSelectedChannels() Returns an array of all selected mixer channels
.getChannelList() Returns the main channels object
.prefixMixerChannels() Adds 3 digit sequential number prefixes to all mixer channels
.removeChannelPrefixes() Removes the prefixes added by prefixMixerChannels()
Mixer
.resetMixer() Resets most of the mixer parts outlined below.
.resetInputGain( [integer] ) Resets input gain to unity. 0=All 1=Selected
.resetPolarity( [integer] ) Resets channel polarity. 0=All 1=Selected
.resetPan( [integer] ) Resets pans to center. 0=All 1=Selected
.fadersToInfinity ( [integer] ) Reset faders to infinity gain. 0=All 1=Selected
.fadersToUnity( [integer] ) Resets faders to unity gain. 0=All 1=Selected
.resetVCA( [integer] ) Removes VCA assignments. 0=All 1=Selected
.resetSpeaker( [integer] ) Resets channel mode to stereo. 0=All 1=Selected
.setInputGain( value [integer] ) Sets the input gain levels
.setFaderLevel( value [string] ) Set fader levels
Window Management
.openWindow( Name [string] ) Opens the target window S1.openWindow( "MessageWindow" );
.closeWindow( Name [string] ) Closes the target window
Messaging
print( Name [string] ) Writes text to the Studio One messages window for debug These are all raw functions, they're not part of the 'S1' object model
alert( Name [string] ) Writes text to a message box
ask( Name [string] ) Asks the user a Yes/No question let msg = "Do you really want to do that?"
if (ask(msg) === 1) return; // selected 'No'
getAllPropertyNames( [object] ) Prints members of an object to the message window getAllPropertyNames(S1) // Show all of the functions in the S1 object
File IO
.writeXML( rootName [string], data [array], url [Host.Url] ) Writes a simple xml file for settings, etc. S1.writeXML( "Settings", "My Folder Name", ["Test1, Value1. Test2, Value2"] ,"settings")
.readXML( rootName [string], items [array], url [Host.Url] ) Reads the simple xml file written by .writeXML() // returns the values from the attributes in an array const values = S1.readXML("Settings", "My Folder Name", ["Test1","Test2"], "settings")
.writeTextFile( folder [string], filename [string], text [string] ) ) Reads the simple xml file written by .writeXML() S1.writeTextFile("My Settings","settings", [string array])
.readTextFile( folder [string], filename [string] ) ) Reads the simple xml file written by .writeXML() // read file into an array const text = S1.readTextFile("My Settings","settings", [string array])
.getLegalFileName( fileName [string] ) Removes any illegal characters from a file name
